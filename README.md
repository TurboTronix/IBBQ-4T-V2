# IBBQ-4T-V2

## How to use
HomeAssistant Lovelace UI for IBBQ-4T V2

1. The ibbq.yaml file into your packages folder
2. The ibbq_lovelace.yaml file into your lovelace folder

Next step is to incorporate Alexa/Mobile notifications
